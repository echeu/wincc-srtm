This is a generic JCOP Framework page that provides basic descriptions of the alarm severities.

Warning An alarm level which indicates an undesired condition. In such a case there is no impact on the physics data quality and no immediate intervention is required.
Error An alarm level which indicates an undesired condition for which action is required. In such a case there is a risk of an unwanted impact on the physics data quality or risk to the detector.
Fatal An alarm condition for which immediate intervention is required.
